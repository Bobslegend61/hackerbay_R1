    Set up

    Install Dependencies

    $ npm install

    Start server

    $ npm start

    Start server in dev mode using nodemon

    $ npm run dev

    Run test

    $ npm test

    API Endpoint

    /login - POST REQUEST Logs in the user. Accepts any username and password and returns an object containing the authorization token.

    Example

    Using postman

    /patch - PATCH REQUEST Send along the authorization token in the header and a body containing the document and the patch operation. it returns an object containing the result of the applied patch to the document.
